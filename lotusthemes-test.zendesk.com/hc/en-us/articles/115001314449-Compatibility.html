Introduction

The visual editor is used in the composition of every Activity form in Salsa Engage. This help section documents the Visual Editor as if it were a stand-alone part of Salsa Engage. However, in practice, the the Visual Editor can be found only in the context of a specific Activity form. This help section covers the common options and standard workflows of the Visual Editor for all Activities.

Many components of the visual editor are featured in all of the Activity forms, however each Activity form has unique elements, and each element has different editing tools. For example, Sign-Up Forms do not have payment amount configuration options since those elements don't apply to signup pages. Unique options per Activity are outlined in more detail in each Activity type's documentation.

Tip: If you don't know how to get to The Basics, go to the list of Activities, and choose the Activity in which you are interested.

Note: Salsa Engage's email tool also has a visual editor. To learn more about the Visual Editor for Email, click here.

Compose/Choose Layout

After you have complete The Basics for your Activity, you'll select a layout. All Activities offer several pre-defined layouts and one Blank layout. Note, it's not possible to swap layouts after one has been selected.

Content Tab

All Activities include a Compose Content and Confirmation step as well as a step for creating Autoresponders. Events and Peer to Peer Events include additional tabs for the creation of Registration and Checkout Pages. Read more about Events here and Peer to Peer Events here.

Views

Salsa Engage automatically lands on the Content tab after a layout has been chosen. There are three tabs under content: Outline, Edit and Preview. Salsa Engage defaults to the Edit View.

Edit View

Most of work will be done from here, as this is where elements and content are added, edited, or removed.

Outline View

The Outline view enables you to shift Rows or Elements in your form. The Outline view is the only place where you can make structural changes. Click, hold, and drag elements into different sections of the form. Click, hold, and drag the icon to move entire sections.

Preview View

The Preview view enables you to she what your form will look like on a desktop, tablet, or mobile phone. Click a screen type at the left of the window.

Rows, Elements, and Design

On the right-hand side of the menu bar that features the View Buttons (Outline, Edit, and Preview), you'll see tabs that you can use to add Elements: Rows, Elements, and Design.

Rows

This tab brings up a menu of drag-and-drop columns that include Header and Footer, but also multi-columns and sidebars. Salsa Engage allows for nested rows up to three deep.

Click the Rows tab, and then drag the row option into the cell that you want to break into rows.

Elements

Elements enable you to add content to Activity forms.

Click the Elements tab, and then click and drag any element into the cell to which you want it added. Note, some elements can be used only once; if one of these elements is already included in your form, it will be grayed out.

Following are all of the elements available in Salsa Engage. (Note: you will see only a subset of these for any given Activity form.)

  • Text - Use this element to enter compelling content.
  • Image - Use this element to apply images. To make edits to images, use Salsa Engage's built-in editor from Adobe. See details here.
  • Divider - Apply a divider to any row here. To change the color, click on the Design tab to see options.
  • Logo - Choose the logo element to paste the organization's logo into a row. To update an existing logo or apply a new one, go to Settings > Switch to > Look and Feel.
  • Button - Apply a button to your form and choose the color, size, text, destination and alignment.
  • Callout - This element functions much like the Text element, however it is designed to call attention to itself using color and border. Edit text, color, and border.
  • Share - Encourage supporters to spread the word via their social networks.
  • Form - This is the area in which supporters enter information. Choose which fields to require depending on the Activity.
  • HTML - Enter your own HTML code if you'd like. Enter HTML here, then choose "Source View" and edit HTML directly. You can also add <style> tags. Another common use is for wrapping text around images. This feature is does not support scripts. If you need to use scripts, use a widget.
  • Map - This element is available only on Events and Peer to Peer Events forms, and it automatically renders based on the location that was entered during the Setup step. If you wish, you can remove it.
  • Location - Like the Map element, Location is set during Setup and will automatically populate on Events and Peer to Peer Events pages. You can edited it in the Visual Editor.
  • Add to Calendar - If applied to an Event or Peer to Peer Event, this will give visitors to those pages the option to click and automatically add the event to their calendars.
  • Register Button - Choose a prominent location for the register button or include one both above and below the fold. Like standard buttons, you can edit the look and feel. In addition to the Register button, you can also add a donate button.
  • Donate Button - Choose a prominent location for the donate button or include one above and below the fold. Like standard buttons, you can edit the look and feel.

In addition to the elements described above, Salsa Engage's Peer to Peer tool has elements not found in standard Events. Read more about them in Peer to Peer documentation.

Design

The tab enables you to define design and page preferences.

Click the Design tab to open the Update Design Basics window, which contains two tabs:

  • Design Preferences - This tab allows for changes to be made for this specific form that may deviate from the default choices made at the account level. Changes can be made to general settings (background colors and dividers), text, buttons, callouts and even form level changes to field styles.
  • Page Preferences - This tab enables you to choose to set a background image and link, apply keywords and a Facebook title, description, and images.

Add and Edit Element Content

When you are on the Content tab and hover over an Element or object within an Element, the Element and/or object will be outlined in orange, and editing tools will appear for that Element and/or object. This section explains those tools.

Row Movement Options

To the upper-left corner of each Element and and each object within an element is a small, orange rectangle with white dots. Click this rectangle to open a menu that enables you to move the Element.

Element Settings, Edit (Pencil Icon), Delete

The options available by clicking any of these icons will depend on the type of design element you're working on. For instance, selecting the Pencil icon on an Image element will bring up your image library. Clicking the same icon on a Text element gives you a text editor.

Following is a brief description of the types of features that are available via the Pencil for each type of Element:

Text - Selecting the Pencil on the Text element element will bring up the Edit Text window. The window feature many of the editing tools that you are familiar with from popular desktop publishing tools, such as Microsoft Word and Google Docs. Hover over any icon in the window to see a description of what it is or just experiment.

Image - For the Image element, the Pencil icon opens the Salsa Engage image library. Using the image library window, you can can add to your form and images to Salsa Engage.

There are no stock images included with your application; you'll need to upload those yourself. You can load images in from your computer via the Upload from Computer tab. Once you've chosen your image, details will be displayed in a frame to the right of the window. From here, you can make the following adjustments to your image:

  • Alignments can be adjusted to the left, right, or center within the Image cell.
  • Alternate text can be applied. When something causes the image to be blocked in a supporters browser, this text will appear instead of the image.
  • You can create a hyperlink on your image to either internal or external pages.
  • In the Caption section, to the bottom of the right frame, you can add, style, and position a caption relative to your image.

Once you've selected your image, you have control over the image's size and alignment in the form itself. A slider will appear in the upper right-hand corner of the Image cell. You can click and drag the slider to make the image larger or smaller.

Button - You can add a button anywhere on your form that links to another page. Once you've selected the Pencil, you can refine the button's attributes: Adjust the link, the positioning, and the size.

Callout - When selecting the Pencil for this field, the Edit Featured Content window will open. It's a lot like the edit text window for the Text element. The important choice for this field, however, is made using the Design tab, in the upper right-hand corner of the visual editor. That's where you can adjust the color of the Callout field itself.

Share (Social) - You can add social messaging, such as Facebook, Twitter, and Google+, to your forms. Select the Pencil to change the arrangement of these option vertically or horizontally and to decide which social media options you'd like to include or exclude. By default, all are chosen. You can also add a message which outlines why it is important that your supporters share this page. You can edit this verbiage with a slightly truncated version of the "Edit Featured Content" window, explained in the Text section above.

HTML - Be very careful when using this field. Use the Pencil if you want to tweak HTML directly. Keep in mind that you can break your form if your HTML is invalid or incorrect in any way.

Everything Else - You will additional Elements depending on the type of form you're building. If you're building a fundraising form, you won't see the "Signatures" icon, but you will see it if you're building a petition. These are all editable with the Pencil, but you'll notice that some are grayed out. A grey icon indicates that the field is already included in the form and another cannot be added.

Row Settings, Duplicate Row, and Delete Row Controls

  • Row Settings - The icon that looks like small, interlocking gears is where you can select a background color, border, padding, and corner style for the given cell.
  • Duplicate Row - The icon that looks like loosely stacked sheets of paper, will replicate the cell and place the new copy directly below the original.
  • Delete Row - The trashcan icon is deletes the given element. You'll be prompted before the element is deleted forever.

The Confirmation View Tab

The Confirmation tab is where you can craft a "Thank you" message page (#1) or choose to redirect to another URL (#2). These are optional, but many organizations like to display a follow-up page as confirmation for form submission. This tab also enables you to publish out to social channels.

This tab features many of the controls documented above. For example, click the Pencil icon to add and edit text. You can see the outline view (although there's not much to move around). You can also define some of the design elements.

Social Sharing

A unique element of this tab is the Social Sharing option. If you have social channels configured, you can open the Social Sharing Configuration window, where you can choose which social media channels you wish to include in your follow-up page.

Once you have your messaging the way you like it, you can save the content using the appropriate button in the lower right-hand corner of the window. In most cases, your form will be completed and ready to publish.

Autoresponders

Autoresponders are email messages that trigger once a form has been submitted. Default autoresponders are built in for each type of Activity and can be used as-is or customized. A Visual Editor that is very similar to the one documented here is used for editing Autoresponder messages.
