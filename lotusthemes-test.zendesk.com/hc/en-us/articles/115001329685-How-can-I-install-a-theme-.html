How can I install a theme?

Andrey Metelsky
February 16, 2017 10:38
Follow

You will be able to find instructions on theme installation in our Online Help Center. You can also contact us via hello@lotusthemes.com for a free quote to assist you with the installation if you face any difficulties.

If you prefer not to spend time installing your theme, you can order an installation service and we will perform the installation for you.

Caution

Note: You will need to provide us with a temporary admin access for installation.
