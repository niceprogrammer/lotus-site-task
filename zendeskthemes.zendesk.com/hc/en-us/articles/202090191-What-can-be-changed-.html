What can be changed?

Michael Reid
December 21, 2014 14:13
Follow

This article will tell you about all possible changes you can make to your theme and how we can help you with it.

The answer is "Everything!"

This option is available to Team, Professional, Enterprise, and Elite customers.

Changes via Customize design panel without any line of code!

The Appearance section of the Customize design panel lets you change the following elements of your Help Center:

  • Name of your Help Center;
  • Logo;
  • Favicon;
  • Main hero image (in case the theme contains it);
  • Language of the Help Center.

  Note

Our themes are not compatible with Zendesk colors and fonts variables due to the small number of them. So changing of theme colors and fonts could be done in the CSS-code instead of using Zendesk variables.

Changes via the code

If you are familiar with HTML and CSS, then it's rather simple to make any other changes you like to the theme. For example:

  • Colors;
  • Fonts;
  • Icons (usually, we use Streamline Icons, but it is possible to use any other ones);
  • Links;
  • logo return URL;
  • Header;
  • Footer;
  • Layout changes, for example, columns number;
  • Anything else...

  Note

In case you change the theme on your own we will be unable to install all the further updates of the theme and support of the theme will not be available.

Changes via Lotus Themes

If you prefer to not spend time on it, we can do all the changes for you. We have the Branding and Customization services.

The Branding service is for basic changes. We can change colors, font, background image and icons. If you have brand's style guide we can use it, or we can take the colors from your main website. Usually, branding takes 3-5 working days.

If you need more complex changes, you can purchase the Customization service. The cost of this service depends on the level of difficulty and is calculated separately in each case. Just share your customization requirements with us, and we will calculate the cost.

We will create a demo Help Center where you will be able to check all the changes. Afterward, we will install the modified theme on your Help Center where you can test everything again but with your real content in the preview mode. Only after that and when you are ready you can publish theme.

  The best way to start is to complete our questionnaire. It will take a couple of minutes and help us to understand your case and recommend the best solution.

You can also contact us via the ticket form or hello@lotusthemes.com
